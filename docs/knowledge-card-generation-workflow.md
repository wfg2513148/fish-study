# 知识点复习卡生成方案

本文档定义 Fish Study 中“知识点复习卡”的生成规则。它适用于家长上传错题照片后，系统根据错题提取相关知识点，并生成学生可直接复习的 PDF。

## 目标

- 根据本次错题提取相关知识点。
- 按知识学习顺序组织复习卡，而不是按原题顺序、照片顺序或错因顺序。
- 每张卡只讲知识点本身，不展示原题、题号、照片来源、错因颜色、诊断依据或训练建议。
- 配图服务知识点记忆，必须由 `gpt-image-2` 根据知识点内容生成，不复用原题配图，不使用 SVG/CSS 占位图冒充。

## 输入与边界

错题照片、原题文字、学生答案、颜色标记和诊断依据只作为内部识别证据。它们可以进入错题归因记录、训练卷生成和答案页，但不得进入学生可见的知识点复习卡正文。

学生复习卡正文禁止出现：

- 照片编号、文件名、照片来源。
- 原题题号、第几题、本题、本次错题等原题定位文字。
- 红色、黄色、蓝色、错因、诊断依据、训练建议、难度梯度。
- 原题场景复述，例如“这道题考查”“根据照片”“题目中”。

## 提取规则

1. 从每个错题 cluster 的 `matched_knowledge` 中读取知识点。
2. 过滤 `待定位` 和低置信度知识点。学生卡片只输出确定知识点。
3. 按知识点标题或稳定知识点 ID 去重。同一知识点被多道错题命中时，只生成一张卡。
4. 内部可以保留来源题目和诊断信息用于统计，但渲染卡片时不得读取这些字段。
5. 卡片集合必须来自本批错题命中的知识点，不得脱离本批错题额外扩展成泛泛章节复习。

## 排序规则

排序优先级：

1. 教材/知识库顺序。
2. `grade -> volume -> chapter -> note` 的章节数字顺序。
3. 同一章节内按知识类型排序：
   - 基础概念和结构。
   - 基本规律、性质、公式。
   - 判断方法和实验方法。
   - 图表应用和综合迁移。
   - 易错辨析。

无法定位顺序的知识点不混入正文，可进入内部待确认材料。

## 卡片结构

每张知识点卡固定包含：

1. 知识点名称。
2. 核心定义或结构。
3. 常用公式、结构关系或判断规则。
4. 解题技巧、实验判断步骤或读图思路。
5. 易混点。
6. 自检问题。
7. 使用 `gpt-image-2` 生成的知识点配图。

卡片不需要讲完整原题，也不需要解释学生为什么错。但卡片必须让学生看完后知道“要点是什么、怎么用、怎么算或怎么判断”，不能只写一句概念说明。

前期生成时，内部可以参考学生出错类型来决定讲解重点：概念不清时强调定义、边界和结构图；漏条件时强调圈条件和判断顺序；计算耗时时强调公式关系和步骤压缩。但学生卡片正文只呈现“复习重点”，不出现错因、诊断、颜色或原题定位。

## 配图规则

配图必须根据知识点内容使用 `gpt-image-2` 生成，不采用原题配图，不使用手写 SVG、CSS 图形或临时占位图。

不要为了绘图而绘图。每个知识点先判断配图价值：

- `required`：结构、过程、关系、实验装置、变量趋势等内容，图解能明显帮助理解。
- `optional`：文字已能讲清，但一张关系图能提高记忆效率。
- `omit`：纯术语记忆或规则背诵，配图容易变成装饰或误导。

如果图片不能清楚体现知识关系，宁可无图，也不能放模糊、泛化或误导的图。无图时不显示“待生成”占位图。

每张图片生成前必须有 `prompt_spec`：`visual_goal`、`must_show`、`must_not_show`、`allowed_labels`、`common_misconception`。生成后必须进行视觉验收，确认图像没有章节节次、题号、来源、诊断词，且准确表达知识关系、中文标签可读、A4 页面不拥挤不裁切。不合格最多重生 1-2 次，仍不合格则删除图片并记录省略原因。

中文标签只能作为图中对象的短标注：放在对应结构、步骤、变量或公式部件附近，用引线或箭头绑定。不要在图片顶部或底部放独立说明条，也不要把卡片正文搬进图片。若加标注后仍不能让学生“看图知意”，说明这张图对知识点支撑不足，应省略配图。

推荐图型：

- 结构图：原子结构、植物器官、根尖、茎横切面等。
- 流程图：实验步骤、判断流程、解题步骤。
- 关系图：公式关系、包含关系、质量分数关系。
- 对比图：物理性质/化学性质、有性生殖/无性生殖、分子/原子/元素。
- 变量图：电压、浓度、温度、时间、体积等变量关系。

图片要求：

- 使用本机 `gpt-image-2` skill：`~/.codex/skills/gpt-image-2/scripts/generate-image.sh`。
- 中文标签。
- 图片和图片说明只使用知识概念名，不出现“第几章”“第几节”“第几课时”或“专题编号”等教材定位文字。
- 少文字，主体说明放在卡片正文中。
- A4 PDF 中可读，不拥挤、不裁切。
- 不复刻原题场景，不出现原题编号或原题选项。
- 视觉、空间、结构、实验、图表、流程类知识点应图文并茂。
- PDF 中必须嵌入本地图片对象，不能只保留图片路径。

## 输出关系

日常错题处理可产生三类材料：

- 学生训练卷：用于做题，不含答案。
- 批改答案页：用于批改和掌握判断。
- 知识点复习卡：用于先补知识点，按知识顺序排列，不暴露诊断信息。

旧的“错题知识点讲解”可以作为内部诊断材料或过渡输出；学生正式复习材料应使用“知识点复习卡”。

## 验收标准

- 同一知识点只出现一张卡。
- 卡片顺序能解释为教材/知识学习顺序，而不是错题顺序。
- 卡片正文不出现照片、题号、错因、依据、训练建议、难度梯度等诊断词。
- 配图不是原题图，且与知识点直接相关。
- 配图为 `gpt-image-2` 生成的本地图片，并已嵌入 PDF。
- 学生不看原题，也能用卡片复述定义、关系、步骤和易错点。
- PDF 已嵌入图片，A4 版面清晰，并复制到 `outputs/codex-session-files/`。
