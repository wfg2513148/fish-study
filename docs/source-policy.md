# 资料使用边界

## 优先级

1. 孩子实际使用的纸质教材和老师作业。
2. 官方或出版社公开的教材目录与课程标准信息。
3. 家长提供的本地教辅/课件资料。
4. 其他网络资料只用于交叉验证，不作为唯一依据。

## 本地保存

- `sources/raw/` 保存原始 ZIP、PDF、图片等资料，不提交 Git。
- `sources/extracted/` 保存解压结果，不提交 Git。
- `sources/inventory/` 保存文件清单和校验值，可以提交。

## Obsidian 写入原则

- 写知识点、易错点、题型方法、学习计划和自编题。
- 不复制整页教材、整段课文、整套教辅题。
- 每条笔记保留来源、年级、册别、学科、章节和可信度。
